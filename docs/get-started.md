# 快速开始

你可以在本页面了解Frozen的一些基本知识

## 前置准备

+ Android10+

+ 设备必须已获取Root权限；[Magisk](https://github.com/topjohnwu/Magisk)[KernelSU](https://github.com/tiann/KernelSU)[APatch](https://github.com/bmax121/APatch)

+ 已刷入[Lsposed](https://github.com/LSPosed/LSPosed)模块

#### 对于Android15+的用户

Android15+可能需要Lsposed IT或其它分支版本

+ [Lsposed IT](https://t.me/LSPosed/287)

+ [Lsposed Irena](https://t.me/lsposed_irena)

> 不建议使用Jingmatrix的Lsposed分支，可能存在未知的问题

## 使用说明

1. 在[Frozen下载站](https://d.anmoc.top)中下载对应的模块，并使用你的Root管理器刷入

2. 在Lsposed中启用Frozen App并勾选推荐的作用域

3. 重启你的手机，打开Frozen App配置并使用即可

## 版本说明

正式版：顾名思义，正式且稳定的版本，推荐更新

Beta版：在每次正式版发版后的小功能更新，可更可不更

Alpha版：测试版，存在不稳定性，可能导致未知的问题 

<font color=red>Use as your own risk.</font>
